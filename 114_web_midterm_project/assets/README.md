# 淡江飯店線上訂房系統

## 專案名稱
淡江飯店線上訂房系統

## 組員
蔡琦浚、蔡依廷

## 專案簡介
本專案為期中小組作業，模擬「飯店線上訂房系統」功能。  
使用者可瀏覽各種房型、依條件篩選、即時計算住宿金額，並於線上填寫訂單完成送出。  
系統具備表單驗證、防重送、收藏與即時回饋功能，讓操作流程更直覺。

## 使用技術
- HTML5（語意化結構）
- CSS3 + Bootstrap 5（RWD 響應式設計）
- JavaScript（DOM 操作、Constraint Validation API、自訂錯誤提示）
- LocalStorage（收藏房型記錄）
- GitHub Pages（網站展示）

## 功能特色
1. **房型瀏覽與篩選**  
   - 依房型類別、是否含早餐、價格上限進行篩選。  
   - 支援收藏房型功能，並可只顯示收藏項目。

2. **即時計價功能**  
   - 使用者輸入入住日期、天數、房間數與人數後，系統自動計算預估金額。  
   - 連住三晚享 95 折，自動加上 10% 服務費。

3. **表單即時驗證**  
   - 檢查姓名、Email、手機、日期等欄位格式。  
   - 不符合格式即顯示紅框與中文錯誤訊息。

4. **防重複送出**  
   - 點擊「送出訂單」後按鈕會變為 Loading 狀態，模擬送出 1 秒。  
   - 成功後顯示綠色成功提示並自動清空表單。

5. **RWD 響應式設計**  
   - 以 Bootstrap 5 排版，支援桌機與手機操作。
