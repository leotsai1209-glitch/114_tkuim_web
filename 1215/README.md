TKU : — 資訊管理系探索任務介面

專案簡介

本專案為一個純前端網頁作品，以「Mission Control（任務控制中心）」為設計概念，結合 antigravity（反重力、未來感）視覺風格，打造一個以 淡江大學資訊管理系 為主題的探索式任務介面。

透過首頁與任務頁的設計，模擬學生在淡江大學中的學習歷程與成長方向，並符合課堂作業「至少兩頁前端頁面」的要求。

使用技術
	•	HTML5
	•	CSS3
	•	玻璃擬態（Glassmorphism）
	•	漂浮動畫（Floating Effect）
	•	深色星空背景
	•	JavaScript
	•	任務狀態切換（完成 / 未完成）
	•	任務篩選顯示
	•	無使用任何前端框架（純前端實作）

頁面說明

1. 首頁（index.html）
	•	主題標題：淡江大學：資訊管理系
	•	簡要介紹淡江大學辦學理念（國際化、資訊化、未來化）
	•	顯示三大探索任務卡片：
	•	校園探索任務
	•	學習成長任務
	•	國際視野任務
	•	提供按鈕導覽至任務頁

2. 任務頁（mission.html）
	•	標題：探索任務面板（Mission Control）
	•	顯示與淡江大學相關的四項任務：
	•	認識淡江大學校園與辦學理念
	•	規劃個人學習與修課方向
	•	培養資訊與數位應用能力
	•	拓展國際視野與跨域能力
	•	功能包含：
	•	任務完成 / 未完成狀態切換
	•	任務顯示篩選
	•	返回首頁功能

任務設計理念

本專案以「任務」作為隱喻，代表學生在淡江大學求學過程中需要完成的不同學習階段。
透過 Mission Control 的介面設計，將學習目標視覺化，提升使用者對校園與學習方向的理解。
視覺與風格說明
	•	antigravity 風格：
使用深色背景、光暈效果與漂浮動畫，營造未來感與科技感。
	•	Mission Control 概念：
借用任務控制中心的意象，作為學習與成長的管理介面。
	•	淡江大學主題：
任務內容皆圍繞淡江大學與資訊管理系學生的學習目標設計。

延伸應用

此專案可延伸為：
	•	淡江大學學生學習歷程展示介面
	•	課程、作業或專題任務管理系統前端原型
	•	後續串接後端登入與權限控制（如 Week12 作業內容）
